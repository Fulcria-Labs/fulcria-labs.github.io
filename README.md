# Fulcria Labs Blog

Golf tech insights and swing analysis tips.

Live at: https://fulcria-labs.github.io

## About

This blog accompanies [Swing Analyzer](https://swing.fulcria.com), an AI-powered golf swing analysis tool.

## Tech Stack

- Jekyll (GitHub Pages native)
- Minima theme

## Adding Posts

Add markdown files to `_posts/` with the format: `YYYY-MM-DD-slug.md`

Required frontmatter:
```yaml
---
layout: post
title: "Your Post Title"
date: YYYY-MM-DD
categories: [golf, swing-tips]
description: "SEO description"
---
```

## Local Development

```bash
bundle install
bundle exec jekyll serve
```
